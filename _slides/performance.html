Case Study

Éclair Cake

A not so simple to make dessert that is a joy to eat. It is such a good sweet that it warranted some copyright lawsuits for the actual creator of this recipe.

Let's do a simple experiment, just try to do a google search for what we need in order to make it and how to make it. Bonus points if you manage to find the website of the acclaimed creators of this really delicious sweet.

If you can not find the exact one, any receipes website will suffice. Just be aware that the performance numbers might vary drastically.

The Results?


  • 20.5 MB Downloaded Resources

  • 8.49s Load Time

  • 619 Requests Made
